Frequently Asked Questions

Apakah dashboard ini menjadi milik saya?

Ya. Dashboard dan data sepenuhnya menjadi milik Anda. Namun, layanan maintenance diperlukan agar dashboard tetap akurat, data bersih, dan KPI selalu relevan mengikuti kebutuhan bisnis.

Apakah dashboard ini real time?

Ya. Setiap kali Anda memasukkan data baru di template, dashboard akan langsung otomatis menampilkan hasilnya (realtime). Perbedaan paket maintenance hanya menentukan seberapa sering data divalidasi & dibersihkan oleh saya (bulanan, 2x per bulan, atau mingguan).

Kenapa saya harus berlangganan maintenance?

Dashboard memang otomatis update, tapi maintenance tetap penting karena:

  • Data sering salah input (tanggal salah, angka minus, duplikat)
  • Bisnis berkembang â†’ KPI bisa berubah & perlu revisi
  • Tanpa maintenance, dashboard rawan error & tidak lagi relevan
  • Dengan maintenance, ada support cepat kalau ada masalah
  • Anda juga akan mendapatkan insight bisnis & rekomendasi sederhana berdasarkan data yang dibersihkan

Kenapa harga dihitung per source data, bukan total baris?

Karena setiap source data membutuhkan proses berbeda: cleansing, template input, dan integrasi ke dashboard. Walaupun total baris terlihat kecil, dua source data berarti dua kali pekerjaan setup.

Apa bedanya add-on dashboard dan source baru?

Add-on dashboard: halaman tambahan dari source data yang sama â†’ cukup Rp1.000.000 per halaman.

Source baru: data berbeda (misalnya penjualan + keuangan) â†’ dihitung setup baru karena perlu cleansing & integrasi lagi.

Kalau saya berhenti berlangganan maintenance, apakah dashboard masih bisa dipakai?

Ya, dashboard tetap bisa Anda akses. Bedanya, tanpa maintenance:

  • Data tidak lagi divalidasi secara rutin
  • Error input tidak akan dibersihkan
  • KPI baru tidak ditambahkan

Artinya dashboard tetap ada, tapi kualitas datanya tidak lagi terjamin.

Apakah saya bisa upgrade atau downgrade paket maintenance?

Bisa. Anda bisa mulai dari Basic lalu upgrade ke Pro/Premium jika kebutuhan bisnis meningkat, atau downgrade jika update tidak lagi sering dibutuhkan.
